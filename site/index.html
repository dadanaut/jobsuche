Jobs – Automatisierte Suche

Diese Platzhalterseite wird beim ersten erfolgreichen Lauf der GitHub Action automatisch durch die generierte Ergebnisliste ersetzt.

Falls du das hier weiterhin siehst, prüfe bitte:

  • Secrets gesetzt (ADZUNA_APP_ID, ADZUNA_APP_KEY)
  • Repository-Variablen (HOME_CITY, RADIUS_KM, …)
  • Workflow in .github/workflows/jobs-daily.yml vorhanden
  • Actions Logs → Fehleranalyse
  • Pages: Source = GitHub Actions
